## 🎨 使用方法

### 快速开始

在应用界面输入以下提示词：

```
画一个 3D 螺旋
```

或点击界面上的 "3D 图形" 示例按钮。

### LLM 生成的典型代码

```python
ax = plt.subplot(111, projection='3d')
t = np.linspace(0, 4*np.pi, 1000)
x = np.cos(t)
y = np.sin(t)
z = t
ax.plot3D(x, y, z, 'blue', linewidth=2)
ax.set_title('3D 螺旋')
```

## ⚠️ 注意事项

### 性能考虑

1. **3D 图形比 2D 更消耗资源**
   - 建议适当减少采样点数
   - 复杂曲面可能需要更长渲染时间

2. **提示词建议**
   - 明确指定 "3D" 或 "立体"
   - 描述具体的 3D 特征

3. **安全性**
   - 所有 3D 代码同样在沙箱中执行
   - 安全限制保持不变
